# Security Policy

## Supported Versions

Security fixes are currently provided for the latest version on `main` and the
latest tagged release.

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Report privately with:

- Subject: `SECURITY: eq-ai-tuner`
- Contact: benjamin@local

Please include:

- Affected component/file
- Reproduction steps or proof of concept
- Impact assessment
- Suggested mitigation (if available)

We aim to acknowledge reports within 72 hours and provide a remediation plan
as soon as practical.

## Disclosure

After a fix is available, maintainers may publish a security advisory and
credit reporters who wish to be acknowledged.
